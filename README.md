# My Personal Website

This is a static personal website starter built as an easy-to-edit first version.

Files:

- `index.html` - page structure and placeholder content
- `styles.css` - layout, motion, glass styling, and responsive behavior
- `script.js` - scroll line progress, reveal animations, nav highlighting, and mobile menu
- `assets/avatar-placeholder.svg` - starter artwork you can replace later

How to use:

1. Open `index.html` in a browser.
2. Replace the sample text with your own story.
3. Swap the placeholder image and colors if you want.
4. Hook the form up to a real backend later if needed.
